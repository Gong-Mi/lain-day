# 摇晃的电车

电车上人熙熙攘攘，空气闷热而粘稠。你被挤在人群中，周围充满了各种各样的声音。

> “——你认识那个明星吗？”
> “——什么？今天外勤安排的是我？”
> “——哦哦哦，我SSR！手机快没电那我先挂了——”

这些声音和你无关，它们像白噪音一样包裹着你。[[IF knows_chisa_name]] // 游戏状态：玩家从姐姐那里听说了“千砂”的名字
但千砂的邮件，依然在你脑海里挥之不去。你感觉头痛欲裂。

你必须做点什么，或者什么都不做。

- [（回应邮件）尝试连接到“连接世界”](action:active_overload)
- [（回应邮件）向Navi询问关于“四方田千砂”的事](action:active_overload)
- [（回应邮件）把这封邮件当成恶作剧，删除它](action:active_overload)
- [........ (静静忍耐)](action:passive_overload)

[[ELSE]] // 游戏状态：玩家没有听说过“千砂”
突然，一个声音在你脑海中响起，它不属于周围任何人：

> “你真的认识lain吗？”

这个声音让你感觉头痛欲裂。

你必须做点什么，或者什么都不做。

- [尝试在脑海中回应“你是谁？”](action:active_overload)
- [试图忽略这个声音](action:passive_overload)
[[ENDIF]]

---
### 事件结果

[[IF action == "passive_overload"]]
// 路线：焦虑症 (Anxiety Disorder)
你的视线开始模糊。周围的噪音和晃动都在远去，你只能听到自己急促的呼吸声。

你熟练地调整着呼吸，闭上眼睛，仿佛已经演练过许多次。
这感觉... 是焦虑症吗？

过了很久，你才在下一站的报站声中，慢慢恢复平静。

[[ELSE]]
// 路线：控制电器 (Device Control)
你终于无法忍受。你不知道是在对车厢里的乘客，还是在对脑海中的千砂——你用尽全力，喊出了那两个字：

> **你：** “闭嘴！”

世界瞬间安静了。

不只是心理上的安静。你注意到，周围乘客手机屏幕的光芒、耳机里漏出的音乐、车厢的电子广告牌... 都在这一瞬间，全部熄灭了。

车厢里的人都用惊恐和怪异的眼神看着你。
[[ENDIF]]

无论过程如何，结果都是一样的。你感觉身体和精神被掏空了。
