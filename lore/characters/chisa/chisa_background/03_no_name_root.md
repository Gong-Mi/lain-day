# 第四层：无名的恐惧 (根源)

这是千砂所有行为的最底层、最私人的动机。它是一种根植于日本文化和她个人史的、关于存在本身的创伤。

### 关键词

*   **原初创伤 (Primal Trauma):** 她人生中经历的第一次“存在被抹除”。
    *   关键词：再婚子女, 忘记姓氏, 更换姓氏。
    *   具体的例子：她可能是“荒川千砂”或“丰川千砂”，但现在是“四方田千砂”。过去的名字“找不到了”。

*   **名字与归宿:** 一个文化层面的核心概念。
    *   关键词：名字 = 身份 = 归宿。
    *   在日本文化中，与“家(ie)”、“户籍(koseki)”相关联，是“人”的凭证。

*   **无缘佛 (muenbotoke):** 飞升Wired后，她所陷入的终极状态——一个不被任何人所知、没有归宿的数字孤魂。

*   **悖论 (Paradox):** 整个故事的核心悲剧。
    *   她为了逃离“名字的牢笼”（社会期望），最终却坠入了“无名的深渊”（存在消解）。

*   **地基:** 这个设定是整个四层叙事结构得以成立的“第一因”和地基，它将所有宏大叙事锚定在了一个女孩最具体的痛苦之上。
